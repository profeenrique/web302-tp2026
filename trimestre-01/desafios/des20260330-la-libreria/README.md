# La Librería

Sitio web de una librería ficticia. 

## Estructura del proyecto

Estructura de ficheros del proyecto:

```
.
├── pages/
├── resources/
│   ├── images/
│   │   ├── donde-viven-los-monstruos.png
│   │   ├── el-soldadito-de-plomo.png
│   │   └── voces-en-el-parque.png
│   └── resultado.png
├── static/
│   ├── css/
│   │   └── styles.css
│   ├── fonts/
│   ├── icons/
│   ├── images/
│   │   ├── donde-viven-los-monstruos.png
│   │   ├── el-soldadito-de-plomo.png
│   │   └── voces-en-el-parque.png
│   └── js/
│       └── script.js
├── .gitignore
├── index.html
└── README.md
```

## Descripción de los archivos

| Archivo / Carpeta | Descripción |
|---|---|
| `pages/` | Carpeta destinada a futuras páginas adicionales del sitio web. |
| `resources/images/` | Carpeta que contiene los recursos necesarios para el sitio web. |
| `static/css/styles.css` | Archivo CSS que contiene los estilos del sitio web. |
| `static/images/` | Carpeta que contiene las imágenes utilizadas en el sitio web. |
| `static/js/script.js` | Archivo JavaScript que incorpora la funcionalidad del sitio web. |
| `.gitignore` | Archivo que excluye archivos y carpetas del control de versiones.|
| `index.html` | Archivo HTML principal del sitio web. |
| `README.md` | Archivo de documentación general del proyecto. |
